E-Commerce Suite (Laravel + React)
A modern, full-stack e-commerce solution built with Laravel 12 and React. The project focuses on a seamless user experience, featuring a dynamic frontend powered by Inertia.js and a robust backend architecture.

Key Features:

Real-time Updates: Integrated Laravel Reverb for live order tracking and stock updates.

Secure Payments: Full Stripe integration supporting one-time payments and automated invoicing.

Advanced Auth: Role-based access control (RBAC) using Spatie Permissions and social login via Socialite.

Performance: Optimized database queries with Eloquent, using Redis for caching and Laravel Queues for background email processing.

Quality Assurance: Covered with Feature and E2E tests using Pest PHP.
